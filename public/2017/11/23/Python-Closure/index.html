什么是闭包

闭包（Closure）是词法闭包（Lexical Closure）的简称，是引用了自由变量的函数。这个被引用的自由变量将和这个函数一同存在，即使已经离开了创始它的环境也不例外。所以，闭包是函数和与其相关的引用环境组合而形成的实体。
